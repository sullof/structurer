Structurer

Shape stories with visual narrative boards

A private workspace for story structure

Structurer helps writers build stories with visual boards, color-coded notes, characters, and narrative phases across built-in and custom structures.

Privacy first

Everything you write stays in your browser local storage. No server-side database is used, no user accounts are created, and no cookies are required for app data.

Portable boards

Export any board as JSON and import it in another browser or computer. Your format is consistent across instances of Structurer.

      Click a question to expand the answer.

      What is a board?

      A board is your main story workspace. Every board uses one narrative structure (for example Hero's Journey, Story Circle, or a custom structure) and renders each phase as a column.

      You can think of a board as a visual planning canvas: you place notes inside phases, reorder phases when needed, and refine the story over time. A board also stores its own phase order, notes, and update history used for merge operations during import.

      What is a story structure?

      A story structure is an organizational framework that helps shape a narrative arc: setup, rising tension, turning points, climax, and resolution. It does not replace creativity, but gives you a map to manage pacing and progression.

      If you want background reading, start with Narrative structure (Wikipedia).

      Which story structures are pre-built in Structurer?

      Structurer currently includes these pre-built structures:

      • Hero's Journey
      • Hero with a Thousand Faces
      • Three-Act Structure
      • Save the Cat
      • Story Circle
      • 7-Point Story Structure
      • Romancing the Beat
      • MICE Quotient
      What are phases and notes?

      Phases are the structure steps (columns). Notes are cards inside those columns. Notes can represent plot points, themes, characters, or custom note types.

      Drag notes with the handle (⋮⋮) to reorder them inside a phase or move them across phases. You can collapse notes for compact reading and open them in edit mode when needed.

      How do I create a note?

      In the board editor, click + in the phase header. Then select the note type from the menu. Character notes open from the character submenu so you can choose an archetype immediately.

      New notes are saved locally and can be moved, collapsed, edited, and resized at any time.

      How do I create custom note types and custom archetypes?

      Use the phase + menu:

      • Define custom note type... to create a new note category and choose its color.
      • Define custom archetype... inside the character submenu to add a new archetype.

      These custom definitions are saved in your browser and become reusable across boards.

      I want to use a structure that is not listed. What should I do?

      Go to dashboard and open Add a structure not listed. Add one phase per row, save, and the new structure will appear in the board creation selector.

      For example, if you want to follow a structure like Freytag's Pyramid (which is not included by default), you can add its phases manually and save it as your own reusable structure.

      How do export, import, and merge work?

      Export a board from dashboard board actions. Import from Import/merge a saved board in dashboard.

      • If the imported board UID is new, Structurer creates a new board.
      • If the imported board UID already exists, Structurer merges notes by note UID.
      • When both sides changed, the most recent update wins.

      Recommended multi-device workflow: do one initial export/import to align IDs between the two copies. After that, repeat export/import for incremental merges.

      If phase order differs between the two versions, merge is blocked and Structurer shows a side-by-side conflict modal so you can align the board manually and retry.

      What are groups and when should I use them?

      Groups let you organize multiple boards into a sequence. This is useful for multi-arc projects, series planning, or alternate structure explorations of the same story.

      Use Create group in the dashboard to name a group and pick boards with checkboxes. To add an existing board to a group later, open that board’s actions and choose Add board to group (available once you have at least one group), then pick the group from the list. Manage order and membership from group actions on the dashboard.
