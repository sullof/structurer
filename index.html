Structurer

Shape stories with visual narrative boards

A private workspace for story structure

Structurer helps writers build stories with visual boards, color-coded notes, characters, and narrative phases across built-in and custom structures.

Privacy first

Everything you write stays in your browser local storage. No server-side database is used, no user accounts are created, and no cookies are required for app data.

Portable boards

Export any board as JSON and import it in another browser or computer. Your format is consistent across instances of Structurer.

      Help

      Core concepts

      • Board: your story workspace built on one structure.
      • Phases: columns of the selected structure.
      • Notes: color-coded cards (plot, theme, character, custom types).
      • Groups: ordered collections of boards for multi-board planning.

      How to add notes

      • In a phase, click + then choose note type.
      • For character notes, choose an archetype first.
      • Drag notes with the handle (⋮⋮) to reorder or move across phases.

      Custom note types and archetypes

      • From the phase + menu, choose Define custom note type...
      • Pick a visual color from the color grid.
      • For custom archetypes, choose Define custom archetype... in the character submenu.

      Custom structures

      • In dashboard, open Create custom structure.
      • Add phase rows and save. The structure becomes selectable for new boards.

      Export, import, and merge

      • Export from board actions in dashboard.
      • Import from Import a saved board.
      • If imported board UID is new, a new board is created.
      • If imported board UID already exists, notes are merged by note UID using most recent update.
      • Best practice for multi-device sync: do one initial export/import first to align IDs, then continue with incremental export/import merges.
      • If phase order differs, merge is blocked and a comparison modal is shown.

      Groups

      • Create a group by opening board actions and selecting Add board to group.
      • Open a group from dashboard to review boards in sequence.
      • Use group actions to reorder boards or remove a board from the group.
